# UtilityFlow — Operator Secrets Template
#
# HOW TO USE:
# 1. Copy this file to .secrets.local.txt:    cp .secrets.template.txt .secrets.local.txt
# 2. Fill in the REAL values in .secrets.local.txt on your local machine.
# 3. NEVER commit .secrets.local.txt — it is in .gitignore.
# 4. When you start a new Manus session, drop .secrets.local.txt into the
#    project root in the sandbox. The agent (per MANUS_HANDOFF.txt §3a)
#    auto-parses this file and calls webdev_request_secrets for each line
#    WITHOUT asking you anything.
#
# FILE FORMAT:
#   - One KEY=VALUE per line (no quotes, no spaces around =)
#   - Lines starting with # are comments and ignored
#   - Empty lines are ignored
#   - Values containing # or whitespace are taken verbatim until newline
#
# SECURITY:
#   - Treat .secrets.local.txt like a password manager export.
#   - Store on encrypted disk; do not email or paste in chat.
#   - Rotate any secret that has ever been pasted in plain chat.
